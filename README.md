# Kubide Code Exercise

## Installation
1. Clone repository
  ```bash
    git clone https://github.com/jcamposs/angular-rocks.git
  ```
2. Install gulp
  ```bash
    npm install -g gulp
  ```

3. Move to the project
  ```bash
    cd angular-rocks
  ```

4. Install dependencies
  ```bash
    npm install
    bower install
  ```

## Usage
 * Start the server
   ```bash
   gulp serve
   ```
 * Run tests
    ```bash
    gulp test
    ```
 * Build the sources
    ```bash
    gulp
    ```
    *after build the sources files will be created in ```dist/``` folder
